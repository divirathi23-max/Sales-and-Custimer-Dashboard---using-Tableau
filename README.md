# 📊 Sales & Customer Dashboard using Tableau

## 📌 Overview
This project presents an interactive **Sales and Customer Dashboard** built using Tableau.  
It demonstrates strong **data analysis, data visualization, and business intelligence skills** by transforming raw data into actionable insights.

The dashboard enables stakeholders to monitor KPIs and supports **data-driven decision-making** through intuitive and interactive visualizations.

---

## 🎯 Objectives
- Perform **exploratory data analysis (EDA)** on sales data  
- Analyze **key business metrics (KPIs)** such as revenue, profit, and quantity  
- Understand **customer segmentation and behavior analysis**  
- Identify **sales trends, patterns, and growth opportunities**  
- Support **business decision-making using data insights**

## 📂 Dataset Description
The project uses structured and relational datasets:

| Table Name | Description |
|-----------|------------|
| Customer  | Customer details (ID, name, segment) |
| Orders    | Transactional data (sales, profit, quantity) |
| Products  | Product categories and sub-categories |
| Region    | Geographic sales distribution |
